Attendance Management System
📌 Project Overview
The Attendance Management System is a web-based application designed to simplify and automate the process of tracking attendance. It provides separate panels for Admin and Users, ensuring efficient management and accurate record-keeping.
- Admin Panel: Allows administrators to manage users, view attendance records, generate reports, and control system settings.
- User Panel: Enables users to mark attendance, view their records, and access personal information securely.
🚀 Features
- Secure login system for Admin and Users
- Real-time attendance marking using AJAX
- Admin dashboard for managing users and attendance data
- Attendance reports and summaries
- Responsive design for easy access across devices
 Technologies Used
- PHP – Server-side scripting for backend logic
- MySQL – Database management for storing user and attendance records
- JavaScript – Dynamic client-side interactions
- AJAX – Asynchronous data updates without page reloads
- HTML5 – Structure of web pages
- CSS3 – Styling and responsive design
